캐릭터 추가 필드 시안

다중 이미지 · 대사 · 색상 팔레트 · 양방향 관계 — 각 기능마다 2~3개 배치안을 담았습니다. 상단 탭으로 기능을 전환하세요. 실제 캐릭터 상세 페이지의 T1 톤·섹션 라벨을 그대로 사용합니다.

이미지 여러 장 — 표정차·의상차·연령대별 러프를 캐릭터당 여러 장 보관하고 대표 1장을 지정합니다. 대표는 카드·관계·연표 썸네일에 쓰입니다.
현재 모델: imageId 하나 → 제안: images: {id, caption}[] + coverImageId
